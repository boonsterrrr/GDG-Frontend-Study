### SPA(Single Page Application)

더 이상 한 페이지에 하나의 정보를 담을 필요가 없어짐. 한 페이지를 하나의 앱처럼 작동하게 하면 됨.
**확장팩**
spa 라이브러리, 프레임 워크들이 생김으로 새로고침 없이 화면이 전환이 가능해짐.

## 라이브러리 vs 프레임워크

라이브러리(=도서관)
: 원하는 기능을 찾아서 쓰면 됨. 도구 상자 같은 거임. 필요하면 골라서 사용하면 됨.
-> 정해진 규칙이 없어서 유연함. 대신 고려할 점이 많음
:'개발자가 코드를 제어'

프레임워크(=틀)
: 강제적으로 정해진 규칙 안에서 사용하면 됨.
-> 규칙 외의 상황에서는 못 사용하지만, 그만큼 일관적으로 사용할 수 있음.
:'프레임워크가 코드를 제어'
